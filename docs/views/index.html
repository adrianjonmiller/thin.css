Thin CSS

This guide shows how to build common UI structures using only the provided utility classes. It does not add any styles beyond your utilities.

{{> nav }}

Getting Started

To get started, include the following CSS file in your HTML:

<link rel="stylesheet" href="https://unpkg.com/thin.css" />

Then, start building your layouts with the provided utility classes.

For example:

<div class="flex--center">
