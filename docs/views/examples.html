Utility CSS – Examples & Recipes

This guide shows how to build common UI structures using only the provided utility classes. It does not add any styles beyond your utilities.

1) Page Frame: Sticky Header, Content, Sticky Footer

Use fixed positioning helpers for header and footer, and add padding on the main content to avoid overlap.

Markup 
<header class="fixed--top fill--width padding--all-md">
  <nav class="flex--between flex--items-center">
    <a href="#home">Brand</a>
    <div class="flex--items-center flex--gap">
      <a href="#docs" class="margin--left-md">Docs</a>
      <a href="#login" class="margin--left-md">Login</a>
    </div>
  </nav>
</header>

<main class="margin--top-xl margin--bottom-xl margin--y-md">
  <article class="margin--x-lg">
    <h1 class="type--size-iv type--weight-700">Page Title</h1>
    <p class="margin--top-sm">Intro copy goes here.</p>
  </article>

  <section class="margin--x-lg">...content...</section>
</main>

<footer class="fixed--bottom fill--width padding--all-md">
  <p class="type--size-i-1">© Company</p>
</footer>

Notes

  • Add margin--top-xl on <main> sized to your header height.
  • Swap to .fixed--bottom-center for a center docked control bar.

17) Accessibility Helpers in Practice

  • Add role, aria-*, and proper landmarks to the structures above.
  • Use type--align-right for numeric columns in tables and receipts.
  • Use overflow--ellipsis with a title attribute to expose full text on hover.
Receipts example 
<div class="reset" role="region" aria-label="Order summary">
  <table class="fill--width">
    <tbody>
      <tr><td>Subtotal</td><td class="type--align-right">$120.00</td></tr>
      <tr><td>Tax</td><td class="type--align-right">$9.60</td></tr>
      <tr><td class="type--weight-700">Total</td><td class="type--align-right type--weight-700">$129.60</td></tr>
    </tbody>
  </table>
</div>

18) Utility Combinations Cheat Sheet

  • Center any box in parent: layout--center inside a layout--container.
  • Center text block: type--align-center margin--auto-y with a fixed width.
  • Sticky CTA bar: fixed--bottom fill--width flex--between padding--all-sm.
  • Full-bleed section: fill--viewportWidth margin--zero-y.
  • Visually limit lines: overflow--ellipsis plus style="max-width:<ch>".
  • Equal spacing in a row: flex--between on container, apply fixed widths to children.
